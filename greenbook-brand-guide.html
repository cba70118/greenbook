01 — Foundation
Brand Story

The Concept

GREENBOOK takes its name from the caddie's yardage book — the worn, dog-eared notebook that every serious player carries. It's the original data center of golf: distances, slopes, wind breaks, pin positions. We are the digital evolution of that edge.

The Personality

We are precise without being cold. Refined without being stuffy. The aesthetic sits between a private member's locker room and a trading floor — serious about data, but rooted in the culture and craft of the game.

The Audience

Serious recreational bettors who love golf. They know the difference between a Stableford and a stroke-play bet. They read shot-by-shot stats. They want an edge, presented with the gravity the game deserves.

The Promise

Never a guess. Never noise. GREENBOOK surfaces only what moves the needle — clean data, sharp odds, and a record of every wager with the discipline of a scratch handicap.

06 — Voice
Tone & Copy
Do — Write This
"McIlroy gained 1.4 strokes on the field from tee-to-green. The number suggests the outright is live."
Precise. Data-anchored. Draws a conclusion without hyperbole.
Don't — Avoid This
"HUGE round incoming! Rory is on FIRE 🔥 don't miss this bet!"
Exclamation marks, emoji, and filler language undermine credibility.
Do — Write This
"−4 thru 9. Strokes gained putting: +1.8. Line moved from +240 to +160."
Mono-style data entries are clear, scannable, and professional.
Don't — Avoid This
"He's been really good lately and the odds look pretty favorable if you ask me."
Vague, casual, and unquantified. Offers no signal.
07 — Usage
Design Rules

Typography Rules

  • Use Cormorant Garamond for all scores, odds values, and hero numbers
  • Use DM Mono for all labels, metadata, odds notation, and timestamps
  • Use Lora for prose copy and editorial descriptions only
  • Never mix mono and serif in the same data cell
  • Italics are editorial — never use italic for data or UI labels
  • Minimum readable type size: 10px mono, 14px serif

Color Rules

  • Brass (#C9A046) is the single accent — use sparingly for hierarchy peaks only
  • Bogey Red is reserved for negative P&L and penalty strokes — never decorative
  • Birdie Blue is data-only — for select chart elements and informational states
  • Never place cream text on the brass accent color
  • Dark backgrounds require the full green ramp — never approximate with generic grays
  • On light surfaces, use green-800 or green-700 for primary text only

Logo Rules

  • Minimum clear space: equal to the height of the "G" in GREENBOOK on all sides
  • Never distort, rotate, or recolor the mark individually from the wordmark
  • The mark may appear without the wordmark at sizes ≥ 32px only
  • Never render the logotype on a competing green that lacks sufficient contrast
  • Brass variant is approved for premium and awards contexts only

Data Display Rules

  • Always show sign prefix on scores and P&L values (+ or −, never bare number)
  • Use unicode minus (−) not hyphen (-) for negative values in display contexts
  • Odds notation: American format −115 / +240; EU decimal where required
  • Score-to-par values always in Cormorant Garamond at the largest available size
  • Live data cells should never truncate mid-update — use skeleton states
